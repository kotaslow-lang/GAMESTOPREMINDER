# Game Stop Reminder

Game Stop Reminder（ゲームストップリマインダー）は、ゲームのプレイ時間を監視し、設定した時間が経過した際にフルスクリーンのポップアップで通知を行うことで、長時間のゲームプレイを防ぎ、適切な休憩を促すためのWindows向けデスクトップアプリケーションです。

## 主な機能

- **プロセス監視**: 指定したゲームの実行ファイル（`.exe`）が起動しているかをバックグラウンドで自動的に検出します。
- **プレイ時間の計測とリマインド**: ゲームセッション開始から時間を計測し、設定した制限時間に達すると画面全体を覆うようなポップアップ通知を表示して休憩を促します。
- **ゲームごとの個別設定**: 監視するゲームのリストを管理し、全体でのデフォルト制限時間だけでなく、ゲーム個別の制限時間を設定することが可能です。
- **システムトレイ常駐**: アプリケーションはシステムトレイ（タスクトレイ）に常駐し、バックグラウンドで動作し続けます。
- **プレイ履歴の記録**: 過去にどのゲームをどれくらいの時間プレイしたかの履歴が自動記録され、アプリ内から確認できます。

## 必要要件

- Windows OS
- Python 3.8 以上

## インストール方法

1. リポジトリをクローンまたはダウンロードします。
2. 必要なPythonパッケージをインストールします。

```bash
pip install -r requirements.txt
```

（主に `PySide6` と `psutil` がインストールされます）

## 使い方

1. `main.py` を実行してアプリケーションを起動します。

```bash
python main.py
```

2. 起動するとシステムトレイにアイコンが表示され、メインウィンドウが開きます。
3. **設定**（歯車アイコンまたは設定ボタン）から、監視したいゲームの実行ファイル名（例: `genshin.exe`、`ffxiv_dx11.exe` など）を追加します。
4. メイン画面の **「検証開始」** ボタンを押すとプロセスの監視が始まります（設定で自動開始をオンにすることもできます）。
5. 設定したゲームを起動すると時間の計測が始まり、制限時間を超えるとリマインダーのポップアップとサウンドで通知されます。

## プロジェクト構成

- `main.py`: アプリケーションのエントリーポイント。全体的な制御とポップアップUIを含みます。
- `game_detector.py`: `psutil` を使用したプロセスの常時監視システム。
- `reminder_manager.py`: 各ゲームのセッション管理と時間計算を行います。
- `ui/`: メインウィンドウ、システムトレイ、設定ダイアログなどのUIコンポーネント。
- `data/`: ユーザー設定とプレイ履歴を保存・読み込みするためのストレージ関数群。
- `style.qss`: アプリケーション全体に適用されるカスタムスタイルシート。
- `requirements.txt`: 依存パッケージリスト。
